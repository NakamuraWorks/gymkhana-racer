Gymkhana Racer

🏁 How to Play

⌨️ Keyboard Controls

  • ← → Steering
  • X Accelerator
  • Z Brake / Reverse

🎮 Gamepad Controls

  • Left Stick Steering
  • A Accelerator
  • B Brake / Reverse

🏁 Race Rules

  • 🟢 Pass the green start/finish line to start
  • 🟡 Pass yellow checkpoints in order
  • Complete all checkpoints to finish a lap
